If copy events are blocked on the page:

Follow these steps to bypass the block and enable the copy event:

  1. Open the developer console:
    • Press Ctrl + Shift + I (Windows/Linux) or Cmd + Option + I (Mac).
    • Go to the Console tab.
  2. Paste this code into the console:
    3. setInterval(() => {
    (function() {
        const originalStop = Event.prototype.stopImmediatePropagation;

        Event.prototype.stopImmediatePropagation = function() {
            if (this.type === "copy") {
                console.log("Bypassed copy event block!");
                return;
            }
            originalStop.apply(this, arguments);
        };
    })();
}, 1000);
  3. Press Enter to run it.

This should allow the extension to work on the page.
